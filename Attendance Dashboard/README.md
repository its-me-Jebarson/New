# Attendance Dashboard

## 📌 Project Overview

The **Attendance Dashboard** is a web-based application designed to manage and track student attendance efficiently. It allows users to add members, mark attendance, search members, and export attendance records. The system provides a simple and user-friendly interface for managing attendance in classrooms or small teams.

---

## 🚀 Features

* Add members with **Name and Register Number**
* Mark attendance as **Present / Absent**
* **Search members** quickly using the search bar
* **Export attendance data to CSV**
* **Remove individual members**
* **Remove all members with confirmation**
* **Notification system** for actions like adding or removing members
* Responsive dashboard UI

---

## 🛠 Technologies Used

* **HTML** – Structure of the web application
* **CSS** – Styling and layout of the dashboard
* **JavaScript** – Functionality and interaction

## ⚙️ How to Run the Project

1. Download or clone the project folder.
2. Open the folder in **VS Code or any code editor**.
3. Open **index.html** in a browser.
4. Start adding members and marking attendance.
